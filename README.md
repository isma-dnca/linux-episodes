# Linux Episodes

This repository contains a series of Linux command explanations as cinematic, story-like episodes.

## Structure

- `episodes/` → Markdown files with Linux command episodes  
- `overthewire/` → Exercises from OverTheWire bandit challenges  
- `assets/` → Images and diagrams for episodes  

## Usage

Read each episode in `episodes/` for step-by-step explanations and mental visualizations of Linux commands.
